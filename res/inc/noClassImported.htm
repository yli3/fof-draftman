Import

You'll first have to 1) export data, and 2) export personal scouting data. Navigate to your leaguedata\LEAGUEID folder and select all CSVs. Draftman will choose which ones to use.

You may also update a current class with new information (interviews, post-draft LFA data, etc) by selecting all except rookies.csv.

Whenever rookies.csv is provided, Draftman will wipe the existing class and replace it. Otherwise, it will update the existing class from the remaining CSVs as needed.

Load
If you have a previously saved .DRAFTMAN draft class file, you may choose the 'Load' option to load it again.
Save

This option lets you save a .DRAFTMAN draft class file, including all notes, grades, and other markings made to the imported draft list. The draft class is automatically saved by the browser as you go, but this lets you archive it.

Note that the browser saves as local site data, which can be cleared or not retained at all depending on your privacy & data settings.
